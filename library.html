Librarie vs Framework ?

Framework
Cadre de travail contraint
Pour :
  • Une seul 'bonne' manière de faire
  • tout le monde fait pareil
Contre :
  • Cadre parfois trop rigide
  • Apprentisage parfois long
Librarie
Outils pour réaliser une tache précise
Pour :
  • Choix de/des outils
  • Liberté
Contre :
  • Trop de liberté
  • Chercher et connaitre les outils prend du temps
