Riot.JS

Type : Librarie
Né le : 22 Janvier 2015
Version : 2.2.4

Slogan : A React-like user interface micro-library



Riot is Web Components for everyone.
Think React + Polymer but without the bloat.
It’s intuitive to use and it weighs almost nothing.
And it works today.
No reinventing the wheel, but rather taking the good parts of what’s there and
making the simplest tool possible.
